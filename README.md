### The app was built with 
- react
- typescript


### How to Start the App
Clone the repo with
- git clone https://github.com/nikolaicholakov25/OutdoorsyTest.git

Open a integrated terminal into the main folder
- cd OutdoorsyTest

Install depencies with
- npm i

And then open the app with 
- npm start
